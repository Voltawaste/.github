# ⚡ Voltawaste

Voltawaste is an IoT-based e-waste management platform focused on smart collection, tracking, and sustainable recycling.

## 🌱 Our Mission
To reduce environmental impact by improving how electronic waste is managed and recycled.

## 🚀 Projects
- Voltawaste App (Mobile)
- Voltawaste Web Dashboard
- IoT Monitoring System

## 🌍 Impact
- Promote sustainable e-waste disposal
- Increase recycling efficiency
- Support circular economy
